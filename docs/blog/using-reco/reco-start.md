### vuepress 설치

1. git clone 하기 : 폴더는 clone한 곳의 레포지토리명
2. 그 폴더의 파워쉘 창을 열고 yarn init -y : package.json 생성
3. yarn add -D vuepress ;vuepress를 devDependency로 추가
4. vscode 열고 npx @vuepress-reco/theme-cli init
// 여러 옵션 작성
5. yarn install
